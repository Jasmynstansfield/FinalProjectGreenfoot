PROJECT TITLE: Catch Me If You Can
PURPOSE OF PROJECT: A fun game!
VERSION or DATE: January 25, 2017
HOW TO START THIS PROJECT: Press 'Run'
AUTHORS: Jasmyn Stansfield
USER INSTRUCTIONS:

Play as Leonardo DiCaprio, and catch the falling Oscars. But watch out for the competion!
Try to avoid the faces of those who have previously won the Oscars that Leo was nominated for.

*Use the left and right arrow keys to move.

Oscar:                +10 points
Missed Oscar:         -1 point
Fellow Competition:   -20 points
